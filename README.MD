## Requirements

* macOS 10.14+
* Lightning-to-USB connection
* libimobiledevice

## Installation

```bash
curl -sL https://raw.githubusercontent.com/zhh4enya/fourtools/main/install.sh | bash
fourtools
```

> The installation script may ask for administrator (sudo) privileges.
